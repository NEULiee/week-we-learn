## Ground Rule

1. **수요일 저녁 ?시** 1시간씩 스터디
2. 각자 공부한 내용을 **md 파일로 정리 후 깃허브에 커밋**
3. 각자 공부한 내용을 **5분 ~ 10분 발표**
4. 발표할 때 질문 적어놓고 끝난 후 질문
5. 발표한 내용 관련하여 추가적으로 아는 내용이 있다면 끝나고 말하기
6. 출처 표시

| :----: | :------------: |
| Upload | 처음 올리는 글 |
| Update |   내용 추가    |
| Revise |   내용 수정    |
| Delete |    글 삭제     |
